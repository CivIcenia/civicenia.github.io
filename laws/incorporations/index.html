Law Incorporation Tracker

Track and manage incorporation of legislative changes into master documents.

About This Page

When the Senate passes an amendment, the master document (typically a .docx file or Google Doc) needs to be manually updated to reflect the changes. This page tracks which changes are still pending incorporation and maintains a log of completed updates.

How to mark a change as incorporated

Option 1: Using the CMS (recommended)

  1. Update the master document — Click the law name below to open the Google Doc/PDF, then edit it to reflect the legislative change(s).
  2. Open the CMS — Go to Law Incorporations Log in the CMS (Data → Law Incorporations Log).
  3. Add an entry to the list — Click the "Add incorporations +" button. For "Change Article Slugs", search by title and select one or more acts that you incorporated. You can select multiple acts if you incorporated several changes at once.
  4. Select the law — Use the "Law Identifier" dropdown to select which law's master document was updated.
  5. Fill in details and save — Set the date/time, enter your name, add optional notes, then click "Publish".

Option 2: Direct file edit (for advanced users)

Edit law-incorporations.yml on GitHub directly. Add entries in this format: 

  - change_slugs:
      - first-act-slug-here
      - second-act-slug-here
    law_slug: PASTE-LAW-SLUG-HERE
    incorporated_at: 2025-11-28 12:00:00 +00:00
    incorporated_by: Your Name
    notes: Optional notes

Tip: Use the "Copy change slug" button on each pending item to get the exact slug needed. Use change_slugs (array) for multiple acts, or change_slug (string) for a single act.

All legislative amendments and repeals have been incorporated into their master documents.
